Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântico)

Nesta frase temos um termo em destaque usando a tag <strong> (semântico)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e pequeno neste parágrafo

Texto deletado

Marcamos um texto como excluído, para indicar que pode ser lido, mas não considerado

Texto inserido

Marcamos o texto como inserido com a tag <ins>

A tag <U> também pode ser usada para sublinhar, mas não é semântica

Texto sobrescrito

Usamos a tag <sup> para textos como 23=8

Texto subscrito

Usamos a ta <sub> para textos subscritos, como H2O
